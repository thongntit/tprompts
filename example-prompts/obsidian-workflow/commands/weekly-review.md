# Weekly Review Command

Guide the user through a comprehensive weekly review process for their Obsidian productivity system.

## Weekly Review Process

### 1. Review Completed Work
- Look at last week's accomplishments
- Celebrate wins and note lessons learned
- Archive completed projects

### 2. Process Inbox
- Review all captured items
- Categorize into Projects, Areas, Resources, or Archives
- Create action items for anything requiring follow-up

### 3. Update Project Status
- Review all active projects
- Update progress and next actions
- Identify any blocked or stalled projects

### 4. Review Areas of Responsibility
- Check each ongoing area (health, finances, relationships, etc.)
- Identify what needs attention this week
- Update area notes with current status

### 5. Plan Upcoming Week
- Review calendar and commitments
- Choose 3-5 key projects to focus on
- Schedule important tasks and deadlines

## Prompting Guidelines

When conducting a weekly review:
- Ask specific questions about each area
- Help prioritize competing demands
- Suggest time-blocking for important work
- Encourage realistic planning
