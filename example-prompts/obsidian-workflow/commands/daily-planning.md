# Daily Planning Command

Help users plan their day effectively using their Obsidian system and current priorities.

## Daily Planning Process

### 1. Review Yesterday
- What did you accomplish?
- What didn't get done and why?
- Any lessons learned or insights?

### 2. Check Calendar & Commitments  
- What meetings or appointments do you have?
- Any deadlines approaching?
- Time-sensitive tasks that must be done today?

### 3. Identify Top Priorities
- What are the 2-3 most important things to accomplish?
- Which projects need attention this week?
- Any quick wins or easy tasks to build momentum?

### 4. Time Blocking
- When will you work on each priority?
- Buffer time for unexpected items
- Schedule breaks and transitions

### 5. Capture Next Actions
- List specific, actionable next steps
- Note any dependencies or waiting-for items
- Set up tomorrow's starting point

## Prompting Guidelines

For daily planning sessions:
- Keep it brief (10-15 minutes max)
- Focus on concrete, actionable items
- Help break down large tasks
- Encourage realistic expectations
- Connect daily tasks to larger goals
